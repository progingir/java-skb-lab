Проектик на Spring Boot для обработки http-запросов и json-данных с глобальной обработкой ошибок

## Что сделано

- **Контроллеры**:
  - `HeadersController` — отображает HTTP-заголовки запроса на странице (`GET /headers`)
  - `JsonController` — принимает JSON (`POST /process`), обрабатывает данные и возвращает JSON-ответ
- **Модели**:
  - `RequestData` и `ResponseData` — классы для передачи данных с вложенными объектами `Info`
- **Обработка ошибок**:
  - `GlobalExceptionHandler` — перехватывает все исключения и возвращает кастомный ответ с кодом 502

## Что происходит

- `GET /headers` — возвращает HTML-страницу с таблицей заголовков запроса.
- `POST /process` — принимает JSON (цена и дата), добавляет ID и возвращает обработанные данные.
- При ошибках — кастомный ответ с кодом 502 и сообщением
