#  Бутько Валерия
#  3 - Dependency Injection

Это приложение представляет собой демонстрацию внедрения зависимостей (Dependency Injection) в Spring Boot
Проект показывает, как использовать Spring для управления зависимостями между компонентами через аннотации
`@Service` и внедрение через конструктор

Приложение включает:
- Интерфейс `MessageService`, определяющий контракт для сервисов отправки сообщений
- Реализации `EmailService` и `SMSService`, которые отправляют сообщения через email и SMS соответственно
- Компонент `MessageProcessor`, который использует оба сервиса для обработки сообщений
- Основной класс `DemoApplication`, который запускает приложение и демонстрирует работу внедрённых зависимостей

При запуске приложение выводит в консоль инициализацию бинов, обработку сообщений и их уничтожение
