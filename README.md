### Аннотация *@SpringBootApplication*
Аннотация *@SpringBootApplication* является ключевой аннотацией в Spring Boot,
которая объединяет несколько других аннотаций для упрощения настройки приложения.
Она используется для маркировки главного класса приложения и активирует автоматическую
конфигурацию Spring, сканирование компонентов и другие функции.

*`@SpringBootApplication`* — это комбинация трех основных аннотаций:

-*@SpringBootConfiguration*  
-*@EnableAutoConfiguration*  
-*@ComponentScan*

### Параметры аннотации *`@SpringBootApplication`*

1. `excludeName`
2. `scanBasePackageClasses`
3. `proxyBeanMethods`
4. `scanBasePackages`
5. `nameGenerator`
6. `exclude`
